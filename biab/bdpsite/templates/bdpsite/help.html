Create a budget site

To create a budget (or spending) site using BIAB, create a new project. A project is a site created with BIAB.

To fill in a project with content, add data packages which contain datasets. Then add visualizations for those datasets.

BIAB's visualization features are powered by OpenSpending. When you add a dataset to a BIAB project, BIAB preprocesses it and uploads it to OpenSpending behind the scenes. You can only create visualizations for datasets that have successfully uploaded to OpenSpending.

Create a project

Make sure you're logged in, then click on your username in the upper right-hand corner of the screen. This will take you to your dashboard.

To create a new project from an existing data package, click on New Project. On the next screen, fill in the URL of your data package's metadata file. If you want BIAB to automatically preprocess and upload the package's datasets, leave Automatically upload datasets? checked. (If not, you can preprocess and upload the datasets yourself later.)

You can also create a new project from scratch by clicking New bare Project, in which case you will be asked to fill in some basic information about the project.

Upload a data package

New data packages can be added to any existing project. Just go to the Data Packages tab on the project's dashboard, click Add a new package, and supply the URL of your data package's metadata file.

Prepare a dataset for visualization

Did you skip the auto-uploading feature when you added your data package? No problem. You can ask BIAB to preprocess and upload your data at any time.

Go to the Datasets tab for your project and find the dataset you wish to preprocess and upload. You will see a row of alerts telling you about three important properties of the dataset:

  • whether the CSV has been preprocessed
  • whether the upload metadata has been generated
  • whether it has been uploaded to OpenSpending

The three blue icons to the right of the dataset's name allow you to carry out these three steps. Simply click on them to watch the step take place right before your eyes.

Create a visualization

If your dataset has been uploaded to OpenSpending, you can create a visualization for it.

Go to your project's Visualize tab and click Add a Visualization. Choose the dataset for the visualization and the type of visualization to use.

BIAB's visualizations work by using "drilldowns" to focus on relevant columns in your data. You must choose at least one column as a drilldown, but you can choose as many as you like—each new drilldown will let the viewer of the visualization refine her view of the data by clicking on the graphic.

To add a drilldown, choose a column from the dropdown list, then hit the "plus" button to the right.

After you're satisfied with the selection of drilldowns, enter a description of your visualization in the "Description" box, and then hit Save.

Put a visualization on a project's front page

Each project can have a "featured visualization" that will be displayed on its front page.

Once you've created a visualization you want to feature, go to the Edit Project tab on your project's dashboard and select a Featured viz from the dropdown list.
